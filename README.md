# Macaroons

Этот проект был создан с помощью [Angular CLI](https://github.com/angular/angular-cli) версии 21.2.11.

## Development server

Чтобы запустить локальный сервер разработки, выполните:

```bash
ng serve
```

После запуска сервера откройте браузер и перейдите по адресу http://localhost:4200/. Приложение будет автоматически перезагружаться при каждом изменении исходных файлов.
